## PysonDiary
* 日記を書くアプリ。その日の記念日や明日の天気を表示することもできる。

## Description
1. 日記を作成（画像1枚登録可能）、編集、削除できる。
2. アクセスしたその日が世間で何の記念日か確認できる。
3. 明日の天気を確認できる。

* PythonでCRUD処理をどう作成するか、APIはどうやって使用するのか、スクレイピングの処理をどうするかを確認する個人的学習のために作成。
